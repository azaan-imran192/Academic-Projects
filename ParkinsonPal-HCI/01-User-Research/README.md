## ğŸ” Key Insights (High-Level)

Key themes that emerged from user research include:
- Difficulty using complex mobile interfaces due to tremors
- High cognitive load caused by fragmented health information
- Need for **simple reminders**, **guided exercises**, and **emotional support**
- Strong reliance on caregivers for routine management

These insights directly informed the **design decisions and feature prioritization** in later phases.
